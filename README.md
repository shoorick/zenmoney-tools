Zenmoney tools
==============

Auxilliary tools to improve [Zenmoney](https://zenmoney.ru/)

Requirements
------------

* Python 3
* [Matplotlib](https://matplotlib.org)

Installation
------------

```bash
python3 -m venv .env  # or replace .env with appropriate name
. .env/bin/activate
pip install -r requirements
```
